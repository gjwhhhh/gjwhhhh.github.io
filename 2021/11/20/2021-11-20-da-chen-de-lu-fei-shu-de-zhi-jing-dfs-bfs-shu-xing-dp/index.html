大臣的旅费

很久以前，T王国空前繁荣。

为了更好地管理国家，王国修建了大量的快速路，用于连接首都和王国内的各大城市。

为节省经费，T国的大臣们经过思考，制定了一套优秀的修建方案，使得任何一个大城市都能从首都直接或者通过其他大城市间接到达。

同时，如果不重复经过大城市，从首都到达每个大城市的方案都是唯一的。

J是T国重要大臣，他巡查于各大城市之间，体察民情。

所以，从一个城市马不停蹄地到另一个城市成了J最常做的事情。

他有一个钱袋，用于存放往来城市间的路费。

聪明的J发现，如果不在某个城市停下来修整，在连续行进过程中，他所花的路费与他已走过的距离有关，在走第x千米到第x+1千米这一千米中（x是整数），他花费的路费是x+10这么多。也就是说走1千米花费11，走2千米要花费23。

J大臣想知道：他从某一个城市出发，中间不休息，到达另一个城市，所有可能花费的路费中最多是多少呢？

输入格式
输入的第一行包含一个整数 n，表示包括首都在内的T王国的城市数。

城市从 1 开始依次编号，1 号城市为首都。

接下来 n−1 行，描述T国的高速路（T国的高速路一定是 n−1 条）。

每行三个整数 Pi,Qi,Di，表示城市 Pi 和城市 Qi 之间有一条双向高速路，长度为 Di 千米。

输出格式
输出一个整数，表示大臣J最多花费的路费是多少。

数据范围
1≤n≤10^5,
1≤Pi,Qi≤n,
1≤Di≤1000
输入样例：
5
1 2 2
1 3 1
2 4 5
2 5 4
输出样例：
135

题解分析

此题有多种做法，下面使用求树的直径+dfs 来解决
题意求图中最远的两点，这里的图是无向图，可以看作树
树中所有最短路径距离的最大值即为树的直径。
如何求树的直径？
两次dfs，从任意点 x 出发，dfs搜索到距离 x 最远的点 y ，
再从 y 出发，dfs搜索到距离 y 最远的点 z。
y 到 z 的距离就是树的直径。

证明可以看别人写的证明 https://blog.csdn.net/forever_dreams/article/details/81051578
此处不做证明
那么解题方法就是这个思路。看下实现的代码
代码做了注释

#include <iostream>
#include <cstring>
#include <algorithm>
#include <vector>

using namespace std;
typedef long long LL;
const int N = 1e5+10;
/*
题目思路是，从一个点出发找到距离这个点最远的点，然后从这个最远的点出发再找一次最远点，两个最远点就构成树的直径
*/
int n;
struct Edge
{
    int id,w;
};
vector<Edge>h[N];
int dist[N];//存储某个点到各个点的距离

void dfs(int u,int father,int distance) //当前遍历的u，上次遍历的这次不要再回去father，到最远边的距离distance距离
{
    
    dist[u] = distance;
    for(int i=0; i<h[u].size(); ++i){
        Edge t = h[u][i];
        int id = t.id, w = t.w;
        if(id!=father){
            dfs(id,u,distance+w);
        }
    }
}

int main()
{
    scanf("%d", &n);
    for(int i=0; i<n-1; ++i){
        int p,q,d;
        scanf("%d%d%d", &p,&q,&d);
        h[p].push_back({q,d});
        h[q].push_back({p,d});
    }
    dfs(1,1,0);
    int maxt = 0, maxk;
    for(int i=1; i<=n; ++i){
        // cout<<dist[i]<<' ';
        if(dist[i] > maxt){
            maxt = dist[i];
            maxk = i;
        }
    }
    // cout<<endl;
    // cout<<maxt<<' '<<maxk<<endl;
    dfs(maxk,maxk,0);
    int maxt2 = 0, maxk2;
    for(int i=1; i<=n; ++i){
        if(dist[i] > maxt2){
            maxt2 = dist[i];
            maxk2 = i;
        }
    }
    // cout<<maxt2<<' '<<maxk2<<endl;
    
    LL ans = 10*(LL)maxt2 + (1+(LL)maxt2)*maxt2/2;
    printf("%lld\n",ans);
    return 0;
}
